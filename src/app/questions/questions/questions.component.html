Instrucciones del Juego

Objetivo del Juego:

Construye frases o coloca palabras correctas a partir de los elementos disponibles para las variables, valores y recomendaciones lingüísticas.

Cada categoría debe completarse correctamente para avanzar al siguiente RNF.

Cómo Jugar:

  • Arrastra las palabras disponibles a las cajas correspondientes.
  • Coloca las palabras en el orden correcto para formar la frase.
  • El puntaje aumenta si aciertas y disminuye si te equivocas.
  • Avanza al siguiente RNF con el botón "Siguiente".

Consejos:

  • Lee la descripción del RNF antes de empezar.
  • Organiza tus ideas antes de colocar las palabras.
  • Puedes reorganizar las palabras si es necesario.
  • Observa el temporizador en la esquina superior derecha. Completa el juego en el menor tiempo posible para un mejor desempeño.
